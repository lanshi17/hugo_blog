## LaTeX常用数学符号大全

本文整理了LaTeX中常用的数学符号，包括关系符号、运算符号、集合符号、逻辑符号、箭头符号和希腊字母等。每个符号都附有LaTeX代码和效果展示，方便大家快速查阅和使用。

### 一、关系符号

| 符号名称     | LaTeX 代码   | 效果 |
| ------------ | ------------ | ---- |
| 因此（所以） | `\therefore` | ∴    |
| 因为         | `\because`   | ∵    |
| 等于         | `=`          | =    |
| 不等于       | `\neq`       | ≠    |
| 约等于       | `\approx`    | ≈    |
| 小于         | `<` 或 `\lt` | <    |
| 大于         | `>` 或 `\gt` | >    |
| 小于等于     | `\leq`       | ≤    |
| 大于等于     | `\geq`       | ≥    |
| 属于         | `\in`        | ∈    |
| 不属于       | `\notin`     | ∉    |
| 包含于       | `\subseteq`  | ⊆    |
| 真包含于     | `\subsetneq` | ⊊    |

### 注意事项

1. **宏包依赖**：  
   部分符号需要加载宏包（如 `\mathbb` 需要 `amssymb`）：

   ```latex
   \usepackage{amsmath}
   \usepackage{amssymb}
   ```

2. **数学环境**：  
   所有符号必须在数学环境中使用，例如：

   ```latex
   这是文本模式：\( \because a = b \therefore c = d \)
   ```

3. **大小写**：  
   希腊字母首字母大写会得到大写符号（如 `\Gamma` → Γ）。
